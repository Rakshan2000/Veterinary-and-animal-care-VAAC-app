# Veterinary-and-animal-care-VAAC-app
The Veterinary and animal care application is a system which is used to maintain and manage a small veterinary hospital and to keep track of all the patients details and appointments . It performs operations like login, signup , doctor information, pet accessories ,hospital information, helpline , first aid information and booking an appointment. It was made for the convenience of the hospital as well as the customer. The main application of this system is the doctors can keep track of the customers appointments and records of their trusted customers and customer can easily know which doctor is available , pet accessories available through this application
